### 담당

#### FE

|        | 담당 업무     |
| ------ | ------------- |
| 이지선 | UI/UX         |
| 김훈   | UI/UX         |
| 조호근 | 카카오 맵 API |

<br/>

#### BE

|        | 담당 업무                                         |
| ------ | ------------------------------------------------- |
| 문동식 | Login 기능(카카오, 네이버), 결제 기능(카카오페이) |
| 양시영 | Boards model, 문서 작업, admin custom, Docker     |
| 안현상 | Boards CRUD, 댓글 기능, follow/following          |
